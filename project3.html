A programming creature making assignment

About this project

For this assignment we had to use Processing for the first time, which is a Java-based programming language.

We had to make a creature that could move and respond to user input.

I made my creature simple by using basic shapes and colors. I wanted it to look cute and friendly.

I learned a lot about programming and how to use Processing. It was a fun challenge to make my creature come to life.

Overall, I am happy with how my creature turned out and I enjoyed the process of creating it.
