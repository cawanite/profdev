A behavior change video assignment

About this project

For this project we had to use our poster assignment as inspiration. We wanted to make a video about the dangers of afterpay.

We used a lot of different shots and angles to make the video more interesting. We also used some special effects to make the video more engaging.

We wanted to make a video that would make people think twice about using afterpay. We hope we succeeded in that.

Overall, this was a fun project to work on and we learned a lot about video production.
